# Vibes

A small collection of mood-based playlist generators.

## Features

- Generate a playlist based on your current mood
- Supports moods: happy, sad, chill, hype
- Outputs a shuffled list of track recommendations

## Usage

```bash
python vibes.py --mood chill
```

## Requirements

- Python 3.8+

## Installation

```bash
pip install -r requirements.txt
```
